Downloads

Stable Releases

You can get stable releases of DownloadDaemon and its clients from the download section at sourceforge.net.
There are binary builds for Windows, as well as source packages.
Ubuntu users might simply want to install DownloadDaemon from our ppa at launchpad. To do so, simply type this command in a shell:
sudo apt-add-repository ppa:agib/ppa && sudo apt-get update && sudo apt-get install downloaddaemon

Nightly Releases

Because of the ever changing One-Click-Hoster websites, we also offer nightly-releases of DownloadDaemon (and soon also of the clients) as source packages and Ubuntu binarys. Not yet as Windows binaries, but that might come some day. You can download the source packages from the SVN tag repository or - for Ubuntu users - install the precompiled nightlies by executing this command:
sudo apt-add-repository ppa:agib/ppa && sudo apt-get update && sudo apt-get install downloaddaemon-nightly
